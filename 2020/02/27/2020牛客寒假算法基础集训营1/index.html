A.honoka和格点三角形

题意：
在N*M的点点状图中，求有多少个三个顶点都是整数且至少有一条边平行于x或y轴的三角形
思路：
先枚举平行，再枚举边长。
当边a平行y轴时：a长为1时，则高为2，a边则有(n-1)*(m-2)种情况，不在a边上的顶点有n*2(*2改变顶点方向)种情况；a长为2时，则高为1，a边有(n-2)*(m-1)种情况，不在a边上的顶点有n*2种情况。
同理边a平行于x轴时，但注意有一种情况就是同时平行于x和y轴，我在考虑a平行x轴时将考虑不在a上的顶点有(m-2)种情况以去重。

C.umi和弓道

题意：
人站在(x0,y0)点拿着弓箭，已知有n个靶子的坐标，求在x或y轴加一道挡板使得能射中的靶子不超过k个，求最短的挡板长度(数据保证人和靶子都不在坐标轴上)。
思路：
最多射中k个，就是要求用挡板挡住n-k个及以上，和人在一个同一个象限的靶子无法挡住，那就找x0*xi<0或y0*yi<0的点,存下两点连线与x轴的交点，这样如果挡板在交点则不可能射中靶子。那么就分别枚举x，y轴，用双指针求包含n-k个点的最短连续长度。

她希望你帮她验证一下。她会给你一个正整数n，让你输出它在迭代过程中，第一次迭代成2的迭代次数。

F.maki和tree

题目描述:
有一天，maki拿到了一颗树。所谓树，即没有自环、重边和回路的无向连通图。
这个树有n个顶点，n-1条边。每个顶点被染成了白色或者黑色。
maki想知道，取两个不同的点，它们的简单路径上有且仅有一个黑色点的取法有多少？
注：
①树上两点简单路径指连接两点的最短路。
②和的取法视为同一种。
思路：
首先考虑路径的种类：
1.中间为黑色点，两边是白色
2.一端为黑色，另一端为白色
考虑将树通过黑点分割将所有白点分块。那么就可以针对这两种情况考虑：
我们枚举黑点，黑点连接点所有白点块内的所有点sum构成情况二，在通过枚举黑点连接白点块的前缀和求第一种情况

I.nico和niconiconi

题目描述：
弹幕中刚开始有很多“nico*1 nico*2”等计数菌，但到后面基本上都是“计数菌阵亡”的弹幕了。
nico也想当一回计数菌。她认为：”nico” 计a分，”niconi” 计b分，”niconiconi” 计c分。
她拿到了一个长度为n的字符串，请帮她算出最大计数分数。
注：已被计数过的字符不能重复计数！如”niconico”要么当作”nico”+”nico”计2a分，要么当作”niconi”+”co”计b分。
思路：
dp。状态转移方程：
$dp[i]=max(dp[i],dp[i-size(nico)]+a,dp[i-size(niconi)]+b,dp[i-size(niconico)]+c)$

J.u’s的影响力

题目描述：
μ’s在九人齐心协力下，影响力越来越大了！
已知第一天影响力为x，第二天影响力为y，从第三天开始，每一天的影响力为前两天影响力的乘积再乘以a的b次方。 用数学语言描述是：
设第i天的影响力为f(1)=x，那么f(2)=y，对于 i>2，$f(i)=f(i-1)f(i-2)a^b$
她们想知道第n天影响力是多少？
由于这个数可能非常大，只需要输出其对1e9+7取模的值就可以了。
思路:
x的幂次变化为:1, 0, 1, 1, 2, 3, 5, 8, 13
y的幂次变化为:0, 1, 1, 2, 3, 5, 8, 13,
21
b的幂次变化为:0, 0, 1, 2, 4, 7, 12, 20
可以发现xy都是斐波那契，以y为基准从第四项起求斐波那契矩阵快速幂的第一次乘积，故qp(g,i-3),y的幂就是g[0][0]，x的幂是y的上一项就是g[1][0] (对于我的板子)。
而b的幂从第五项起都是前两项之和加1。可以构造出来这样一个矩阵 (4,2,1)分别对应5，4，3项
I= { 4,0,0,
  2,0,0,
  1,0,0}

g={ 1,1,1,
  1,0,0,
  0,0,1}
g*i得g[0][0]得到第六项7
故qp(g,i-5)
